# What is Genesis Locker?

Genesis Locker is a decentralized liquidity and token locker. Projects can lock LP tokens, normal ERC20/BEP20 tokens, cliff unlocks, and vesting schedules, then share public proof pages.

The platform supports Ethereum, Base, and BSC. It is designed for communities, explorers, bots, DEXTools, DexScreener, and other public integrations.

Genesis Locker is proof infrastructure. It is not an endorsement system and does not guarantee that a token or project is safe.
