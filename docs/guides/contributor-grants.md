# Contributor Grants

Genesis Locker can fund open-source contributors from the community multisig fee share.

Grant categories:

- Contract testing and audits
- API and indexer improvements
- Frontend features and accessibility fixes
- Documentation, examples, and translations
- Chain, DEX, explorer, and wallet integrations
- Community support, tutorials, and launch operations

Grant requests should include the contributor, deliverable, requested amount, payment address, review criteria, and target date.

Payments should be made after review unless the community multisig approves a milestone or upfront bounty. Every paid grant should appear in a monthly treasury report.
