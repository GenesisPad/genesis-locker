# Treasury Reports

Genesis Locker should publish a monthly treasury report for the community multisig.

Each report should include:

- Reporting month
- Community multisig address
- Opening balance by asset
- Fees received by chain
- Grants and operating expenses paid
- Closing balance by asset
- Pending approved grants
- Notes on fee changes, recipient changes, or signer changes

Reports should live in the `treasury` folder and use the month format `YYYY-MM.md`.
