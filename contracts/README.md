# Genesis Locker Contracts

Solidity contracts for Genesis Locker, an open-source liquidity and ERC20/BEP20 token locker.

## Commands

- `npm run build`
- `npm test`
- `npm run deploy:ethereum`
- `npm run deploy:base`
- `npm run deploy:bsc`
- `npm run export:abi`

Each network deploy uses constructor configuration for the chain-specific creation fee.
